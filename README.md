# Komorebi (木漏れ日)

> Virtual Cozy Room / Study Room — ห้องส่วนตัวบนเว็บที่ทำให้คุณโฟกัส ผ่อนคลาย และ productive

**Komorebi** (木漏れ日) แปลว่า "แสงแดดที่ลอดผ่านใบไม้" — สื่อถึงความอบอุ่น สงบ และ aesthetic

## Quick Start

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **Framework**: Next.js 15 + React 19 + TypeScript
- **Styling**: Tailwind CSS 4 + Framer Motion
- **Canvas**: PixiJS (pixel art rendering)
- **Audio**: Howler.js (ambient sound)
- **Auth**: NextAuth (Google + Magic Link)
- **DB**: PostgreSQL (Prisma ORM)
- **Payment**: Stripe / Omise
- **Realtime**: Liveblocks (multiplayer)
- **Hosting**: Vercel + Supabase
