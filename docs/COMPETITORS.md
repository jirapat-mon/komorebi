# Komorebi — Competitor Analysis

## Direct Competitors

### 1. Lofi Girl (lofi.co)
- **What**: YouTube channel + website with lo-fi music player
- **Users**: Millions (YouTube 13M+ subs)
- **Monetization**: Merch, music licensing
- **Strengths**: Massive brand recognition, iconic character
- **Weaknesses**: ไม่มี customization, ไม่มี productivity tools, passive experience
- **Our Edge**: Interactive room + timer + todo + multiplayer

### 2. Forest App
- **What**: Focus timer ที่ปลูกต้นไม้
- **Users**: 10M+ downloads (mobile)
- **Monetization**: Paid app ($3.99) + in-app purchases
- **Strengths**: Simple, gamified, proven model
- **Weaknesses**: แค่ timer, ไม่มี ambient sound, ไม่มี room, mobile only
- **Our Edge**: Full room experience + sounds + web-based + multiplayer

### 3. Noisli
- **What**: Ambient sound mixer + text editor
- **Users**: ~500K users
- **Monetization**: Subscription ($10/month)
- **Strengths**: Clean UI, good sound library
- **Weaknesses**: ราคาแพง, ไม่มี visual room, ไม่มี gamification
- **Our Edge**: Visual room + cheaper + gamification + Thai payment

### 4. Habitica
- **What**: RPG gamified to-do list
- **Users**: 4M+ users
- **Monetization**: Subscription ($5/month) + gems
- **Strengths**: Deep gamification, community, long-running
- **Weaknesses**: ซับซ้อนเกินไป, UI ไม่ modern, ไม่มี ambient/room
- **Our Edge**: Simpler, aesthetic, cozy (not competitive RPG)

### 5. StudyStream
- **What**: Live study-with-me video rooms
- **Users**: Growing (100K+ Discord)
- **Monetization**: Donations, Premium rooms
- **Strengths**: Real people, accountability
- **Weaknesses**: ต้องเปิดกล้อง, heavy bandwidth, no customization
- **Our Edge**: No camera needed, pixel art privacy, customize room

### 6. Lifeat.io
- **What**: Virtual study spaces with ambient video backgrounds
- **Users**: Growing (viral TikTok)
- **Monetization**: Premium subscription
- **Strengths**: Beautiful video backgrounds, Pomodoro, sounds
- **Weaknesses**: ไม่มี customization, ไม่มี multiplayer, video = heavy
- **Our Edge**: Pixel art (lighter), full customization, items, multiplayer
