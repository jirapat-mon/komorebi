## User Journey Map

```
Awareness          Interest           Trial              Convert            Retain
─────────────────────────────────────────────────────────────────────────────────
เห็นบน TikTok  →  เข้าเว็บ       →  ลองใช้ฟรี     →  จ่าย Premium  →  ใช้ทุกวัน
เห็นบน Twitter →  ดู room สวยๆ   →  สมัคร Google   →  อยากได้ของ    →  Streak!
เพื่อนแชร์      →  กดลิ้งค์       →  แต่งห้อง       →  อยากได้เพิ่ม   →  ชวนเพื่อน
Reddit post    →  อ่าน review    →  เปิด Pomodoro  →  ต้องการ stats  →  Seasonal
```
